# AIxie
AI写小说软件 AI一键生成网文小说 AI写网文神器

填上之后点一下这个配置就算启用了，然后就可以开始创作小说。

并且提示词还支持标签化，和模拟多轮对话！！让写作和上下文精准度加强加高！！

我一般选混合，就是传前文梗概和前面几章，前文梗概我一般全传，前面章节数的话我选5。这样写出来的连贯性也高，消耗的输入token我觉得适中。

小编试了一下，非常好用，不过有时候就是需要修改提示词什么的。

贴到系统提示词里面之后，后面新开的小说就会自动用这个提示词里面的内容，去生成我要的内容。
